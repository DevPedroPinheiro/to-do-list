# 📝 To-Do List — Laravel 13

Aplicação de lista de tarefas desenvolvida com Laravel 13 e PHP 8.4, com funcionalidades completas de CRUD, toggle de status e validação de formulários.

## ✨ Funcionalidades

- ✅ Criar, editar e deletar tarefas
- ☑️ Marcar tarefas como concluídas ou pendentes
- 📅 Data de vencimento para cada tarefa
- 📝 Título e descrição por tarefa
- ⚠️ Validação de formulários
- 💬 Mensagens de feedback após cada ação

---

## 🛠️ Tecnologias utilizadas

- **PHP** 8.4
- **Laravel** 13.1.1
- **MySQL** (banco de dados)
- **Bootstrap** 5.3 (interface)
- **Blade** (template engine)
- **Eloquent ORM** (acesso ao banco)

---

## 📚 O que aprendi neste projeto

- Criação e execução de **migrations**
- Uso do **Eloquent ORM** para consultas ao banco
- Estruturação de **Controllers** com Resource
- Definição de **rotas** com `Route::resource`
- Criação de **views** com Blade templates
- **Validação de formulários** com Request
- Uso de **flash messages** para feedback ao usuário
